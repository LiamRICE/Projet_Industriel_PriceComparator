Cloud training
Looking to build or enhance your cloud skills to move your career forward? The IBM Center for Cloud Training offers multiple options for role-based courses that are self-paced or instructor-led for all skill levels. Our role-based courses include:
– Cloud Architect: Gain the skills and knowledge required to begin a career as an IBM Cloud Professional Architect. The exercises include the activities required for designing a secure, cloud-native microservices architecture.
– Cloud Developer: Learn how Cloud Developers build cloud services and large-scale business applications that run in complex cloud environments. After completing this curriculum, you should be able to perform cloud development and operations (DevOps) functions using IBM Cloud services and APIs, which include Artificial Intelligence, cloud databases and cloud security.
– Cloud Site Reliability Engineer: This interactive learning path features practice exercises and hands-on labs in a virtual lab environment. Develop your skills and knowledge around incident management, monitoring, troubleshooting, operations, deployments and security and compliance, and learn the tools needed to manage enterprise workloads in IBM Cloud environments.

Additional tranining options are also available for specialties in Security, Financial Services, Satellite and DevSecOps.

Since April 12, all training and certification courses offered through the IBM Center for Cloud Training will be available at no charge to anyone, anywhere.
